## üéØ What You Will Learn

We implement a **numerical integration algorithm (Euler‚Äôs Method)** to simulate the ascent profile of a multi-stage rocket. Key topics covered include:

### 1. Atmospheric Modeling
Coding the exponential decay of air density (œÅ) and calculating its significant effect on aerodynamic **Drag** throughout the flight.

### 2. Propulsion Physics
Simulating the variable **Thrust** and Specific Impulse ($I_{sp}$) of the Merlin 1D engines as they transition from Sea Level to the vacuum of space.

### 3. Flight Dynamics
Calculating critical flight parameters over time, including:
* Dynamic Pressure (**Max-Q**)
* Acceleration (G-force)
* Mass depletion due to fuel consumption.

### 4. Guidance & Control
Implementing the logic for the **"Gravity Turn"** maneuver, essential for efficient orbital insertion.

### 5. 3D Visualization
Creating stunning 3D trajectory plots and analyzing flight telemetry data directly within MATLAB.
